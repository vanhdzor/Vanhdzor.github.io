Tablas en HTML

Tabla simple con separación de borde de celda


Las tablas en HTML son estructuras utilizadas para organizar y presentar datos en filas y columnas. Los elementos principales de una tabla incluyen table, que es el contenedor de toda la tabla; tr, que define una fila; th, que representa las celdas de encabezado, generalmente en negrita; y td, que define las celdas de datos. Otros elementos importantes son caption, que proporciona un título a la tabla, thead, que agrupa los encabezados, tbody, donde se organizan los datos, y tfoot, que se utiliza para el pie de la tabla, comúnmente para totales o resúmenes.

Tablas


Un ejemplo de estructura de tabla podría incluir filas de encabezados y datos, y un pie de tabla para cálculos finales. Además, atributos como border (para definir el ancho del borde) o colspan (para hacer que una celda se extienda sobre varias columnas) son útiles para el control visual. Aunque estos atributos pueden aplicarse directamente en HTML, es recomendable usar CSS para el estilo y la personalización de las tablas, mejorando así la presentación y funcionalidad de los datos.
